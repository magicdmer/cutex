Ausführliche Beschreibung

Die Klasse QxTreeItemModel stellt ein Model zur Implementierung von Baumstrukturen bereit.

QxTreeItemModel ist ein Model zur Realisierung von Baumstrukturen und arbeitet optimal mit dem Steuerelement QTreeView zusammen. Die Klasse stellt alle wichtigen Funktionen, einschließlich Drag&Drop-Unterstützung, bereit.
