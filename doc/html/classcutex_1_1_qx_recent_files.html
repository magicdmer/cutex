Ausführliche Beschreibung

Die Klasse QxRecentFiles verwaltet eine Liste zuletzt verwendeter Dateien.

Die Klasse QxRecentFiles wird von QxMainWindow zur Verwaltung der Liste zuletzt verwendeter Dateien verwendet. Eine Instanz dieser Klasse kann dem Hauptfenster zugewiesen und mit einem Menü verbunden werden. Die Aktualisierung des verbundenen Menüs erfolgt automatisch.
