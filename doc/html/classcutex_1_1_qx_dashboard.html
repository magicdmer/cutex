Ausführliche Beschreibung

Mit der Klasse QxDashboard kann eine Einstiegsseite für die Anwendung realisiert werden.

Das Dashboard verfügt über Symbole zur Neuanlage und zum Öffnen von Dateien. Wenn dem Dashboard ein Objekt der Klasse QxRecentFiles zugewiesen wurde, wird zusätzlich die Liste der zuletzt verwendeten Dateien angezeigt.
