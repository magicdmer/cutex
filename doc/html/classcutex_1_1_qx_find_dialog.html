Ausführliche Beschreibung

QxFindDialog ist ein fertiger Dialog zum Suchen und Ersetzen von Text.

Dialoge vom Typ QxFindDialog stellen die GUI für eine Suchen/Ersetzen-Funktion von Text bereit.
