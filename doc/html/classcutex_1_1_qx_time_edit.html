Ausführliche Beschreibung

Die Klasse QxTimeEdit stellt ein Widget zur Eingabe einer Uhrzeit bereit.

Ein Widget der Klasse QxTimeEdit ermöglicht die Eingabe einer optionalen Uhrzeit. Die Uhrzeit kann durch betätigen der Backspace-Taste gelöscht werden.
