cutex

Partsoft cutex ist eine Klassenbibliothek für den Qt-Framework, welche die Programmierung erheblich vereinfacht. cutex ist kompatibel zu Qt ab Version 5.5.1 und steht unter der LGPL-Lizenz, d.h. die Bibiothek darf auch für kommerzielle Projekte frei verwendet werden. Bitte beachten Sie das bei Verwendung der Bibliothek, gemäß den Bestimmungen der LGPL, ein Hinweis auf die Quelle (https://www.partsoft.de) erforderlich ist.

Das Projekt entstand ursprünglich durch die Auskopplung einzelner Klassen aus unserer kommerziellen Software Partsoft Money 2016, mit dem Zweck der Wiederverwendung. Daraus entstand mit der Zeit eine komplette Bibliothek, auf der mittlerweile sämtliche Produkte von Partsoft basieren. Auch in Zukunft werden wir cutex weiterhin einsetzen und somit ständig weiter entwickeln.

Beispiele zur Verwendung der Bibliothek finden Sie im Ordner samples, eine detailierte Beschreibung der einzelnen Klassen können Sie dieser Dokumentation entnehmen.
