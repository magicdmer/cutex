Ausführliche Beschreibung

Die Klasse QxSqlTableView stellt eine View für SQL-Modelle bereit.

Die Klasse erweitert die Qt-Klasse QTableView um spezielle Methoden für die Zusammenarbeit mit SQL-Modellen. Außerdem verfügt die Klasse über ein internes QSortFilterProxyModel, wodurch der Verwender kein zusätzliches Model für die Sortierung vereinbaren muss.
