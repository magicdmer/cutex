Ausführliche Beschreibung

Die Klasse QxMainWindow vereinfacht die Implementierung von Hauptfenstern.

Bei Verwendung der Klasse QxMainWindow erübrigt sich die Implementierung vieler in Hauptfenstern häufig benötigten Slots, z.B. für das Auslösen oder ent-/sperren von Menüaktionen. Die automatische Verwaltung, bzw. Aktualisierung des Fenster-Menüs vereinfacht die Programmierung von MDI-Anwendungen. Weiterhin stellt die Klasse Methoden zum Laden und Speichern von Einstellungen, sowie der Liste der zuletzt verwendeten Dateien zur Verfügung.

◆ setupMdiArea()

void QxMainWindow::setupMdiArea ( QMdiArea *  mdiArea,
QMenu *  windowMenu = nullptr 
)

Initialisiert den MDI-Bereich mdiArea. In MDI-Anwendungen muss diese Methode im Konstruktor des Hauptfensters aufgerufen werden, damit die MDI-Schnittstelle korrekt funktioniert. Wenn das Hauptfenster über ein Fenster-Menü verfügt, kann der Zeiger im Parameter windowMenu übergeben werden. Die entsprechenden Menüaktionen werden dann automatisch eingefügt, bzw. aktualisiert.
