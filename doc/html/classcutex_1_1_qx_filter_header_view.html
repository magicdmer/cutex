Ausführliche Beschreibung

Die Klasse QxFilterHeaderView stellt eine Suchleiste für Views zur Verfügung.

Die Klasse QxFilterHeaderView stellt einen horizontalen Header mit einer Suchleiste für Views bereit. Für jede Spalte kann ein Suchbegriff eingegeben werden. Das Projekt filterheader (im Ordner samples) ist ein Beispiel zur Verwendung.
