Ausführliche Beschreibung

Die Klasse QxDoubleEdit stellt ein Widget zur Eingabe von Gleitkommazahlen bereit.

QxDoubleEdit ist eine Erweiterung der Qt-Klasse QLineEdit, welches nur die Eingabe von Gleitkommazahlen erlaubt. Das Widget wurde als Alternative zu QDoubleSpinBox implementiert, da dieses unter dem Betriebssystem Android nicht richtig dargestellt wird.
