Ausführliche Beschreibung

Die Klasse QxWizard vereinfacht die Implementierung von Assistenten.

QxWizard erweitert die Klasse QWizard um eine Sammlung virtueller Methoden, mit deren Hilfe sich das Implementieren vieler Slots erübrigt. Zudem stellt die Klasse einen speziellen Mechanismus zur Validierung von Eingabefeldern bereit, bei dem ungültige Eingaben farblich hervorgehoben werden, ohne den User am weiterarbeiten zu behindern.
