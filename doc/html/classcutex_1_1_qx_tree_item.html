Ausführliche Beschreibung

QxTreeItem stellt die Elemente für das Model QxTreeItemModel zur Verfügung.

Ein Objekt der Klasse QxTreeItem entspricht einem einzelnen Element des Models QxTreeItemModel. Einem Element können beliebig untergeordnete Elemente hinzugefügt werden, wodurch eine Baumstruktur für QTreeView realisiert werden kann.
