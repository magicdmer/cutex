Ausführliche Beschreibung

Die Klasse QxIntEdit stellt ein Widget zur Eingabe von Ganzzahlen bereit.

QxIntEdit ist eine Erweiterung der Qt-Klasse QLineEdit, welches nur die Eingabe von Ganzzahlen erlaubt. Das Widget wurde als Alternative zu QSpinBox implementiert, da dieses unter dem Betriebssystem Android nicht richtig dargestellt wird.
