Ausführliche Beschreibung

Die Klasse QxDateEdit stellt ein Widget zur Eingabe von Datumswerten bereit.

Ein Widget der Klasse QxDateEdit ermöglicht die Eingabe optionaler Datumswerte, z.B. für Geburtstage in einer Adressenverwaltung. Das Datum kann durch betätigen der Backspace-Taste gelöscht werden.
