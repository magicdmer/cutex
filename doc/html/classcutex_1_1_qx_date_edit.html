Ausführliche Beschreibung

Die Klasse QxDateEdit stellt ein Widget zur Eingabe von Datumswerten bereit.

Ein Widget der Klasse QxDateEdit enthält neben einem gewöhnlichen QDateEdit zusätzlich eine Checkbox, mit der das Datum deaktiviert werden kann. Dadurch kann eine Eingabe optionaler Datumswerte realisiert werden, z.B. für Geburtstage in einer Adressenverwaltung.
