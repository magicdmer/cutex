Ausführliche Beschreibung

Die Klasse QxDialog vereinfacht die Implementierung von Dialogen.

QxDialog erweitert die Klasse QDialog um eine Sammlung virtueller Methoden, mit deren Hilfe sich das Implementieren vieler Slots erübrigt. Zudem stellt die Dialogklasse einen speziellen Mechanismus zur Validierung von Eingabefeldern bereit, bei dem ungültige Eingaben farblich hervorgehoben werden, ohne den User am weiterarbeiten zu behindern.
