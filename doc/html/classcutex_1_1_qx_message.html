Ausführliche Beschreibung

QxMessage ist eine Container-Klasse zum Sammeln von Meldungen.

Mit Hilfe der Klasse QxMessage können bei größeren Operationen Meldungen (Statushinweise, Fehlermeldungen, etc.) gesammelt und im Anschluß an den Benutzer ausgegeben werden.
